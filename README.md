### Wiki Agent 
An Interactive AI Agent That Searches Wikipedia For Info Asked

### Get Started 
1. Clone repo 
2. Create virtual environment `python3 -m venv .venv`
3. Activate virtual environment `source .venv/bin/activate`
4. Install packages `uv sync`
5. Create `.env` file and add your `OPENAI_API_KEY` variable
6. Run `uv run mcp_client.py`
7. Converse. To exit chat, type `exit`
